# 🎓 ITM Dehradun Chatbot Assistant

A simple web-based chatbot designed to provide quick information about ITM Dehradun.
Built using **HTML, CSS, and JavaScript**, this project simulates a basic AI assistant with predefined responses.

---

## 🚀 Features

* 💬 Interactive chatbot UI
* ⚡ Instant responses based on user input
* 🎯 Predefined queries (courses, fees, admission, etc.)
* 🧹 Clear chat functionality
* 🎨 Clean and modern UI design

---

## 📁 Project Structure

```
├── index.html   # Main HTML file
├── style.css    # Styling file
├── script.js     # Chatbot logic
```
